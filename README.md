# taskmasterpro-fullstack
Full-stack TaskMaster Pro (Angular + Tailwind, dark mode, filters, Kanban) with .NET 8 API (JWT, EF Core, error middleware). Includes tests, example configs, zero-cost deploy path (Vercel frontend, Railway API + Postgres), optional AWS Lambda + API GW demo, IaC refs, and CI-ready (lint/build/test).
